# 🩺 My Medi Buddy Website

## 📌 Overview
My Medi Buddy Website is a simple, app-style tester landing site for the **My Medi Buddy app**.

It is designed to help early testers understand the app, learn how to use it, and access support features like bug reporting and testing guidance.

The goal is to create a **mobile app-like experience in the browser**, not a basic static website.

---

## 📁 Project Structure

### index.html
Main landing page that introduces the My Medi Buddy app and acts as the entry point for users.

### download.html
Download or access page for testers. Provides instructions or access to the app.

### testing-guide.html
A step-by-step guide explaining how to properly test the app and what testers should focus on.

### report-a-bug.html
A support/info page that explains how feedback works.  
All issues and suggestions are submitted **directly inside the My Medi Buddy app**, not through this website.  
This page exists to demonstrate that in-app reporting system.
