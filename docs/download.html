Installation Steps

  1. Download the APK

    Tap the download button above to save the file.

  2. Allow installation from unknown sources

    Go to Settings → Security → Enable "Unknown sources" OR tap "Settings" → "Allow" when prompted.

  3. Install the app

    Open the downloaded file and tap "Install".

  4. Start testing!

    Find My Medi Buddy in your app drawer and open it up.

Having trouble?

  • App won't install? Check that unknown sources is enabled and you have enough storage.
  • Getting "App not installed"? Try uninstalling any previous version first.
  • Need a hand? Use the in-app feedback form to ask questions.
