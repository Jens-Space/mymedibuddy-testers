Installation Steps

  1. Tap the Download APK button

    Google Drive will open. If prompted, select your Google account to continue. The APK file will start downloading to your phone.

  2. Allow installation from unknown sources

    When Android asks, tap Settings then enable Allow from this source. This is required for any app installed outside the Play Store and is safe to do.

  3. Open the Package Installer

    Once downloaded, Android will show an "Open with Package Installer" prompt — tap it. Then tap Install on the next screen.

  4. Start testing!

    Find My Medi Buddy in your app drawer, open it up, and explore. Don't forget to use the in-app feedback form to share what you find.

Having trouble?

  • App won't install? Check that unknown sources is enabled and you have enough storage.
  • Getting "App not installed"? Try uninstalling any previous version first.
  • Need a hand? Use the in-app feedback form to ask questions.
