Send Feedback 💬

Found something or have an idea? I'd really love to hear from you 💜

Your feedback helps shape My Medi Buddy and make it better over time.

Use the Feedback Form

The easiest way to send feedback is using the form below.

  1. Tap the button to open the feedback form
  2. Fill in a few quick details
  3. Submit — it comes straight to me

You can include screenshots or extra details if you'd like, but it's completely up to you.

What to Include (Helpful but Optional)

  • What happened? A quick description of the issue or idea
  • What were you doing? What led up to it
  • What you expected What should have happened instead
  • Screenshot Always helpful, but not required

What Happens Next

Everything you send comes directly to me. I review feedback regularly and use it to:

  • Fix bugs
  • Improve features
  • Make the app feel smoother and easier to use

Updates are released regularly, so you should see improvements over time.

How Feedback is Prioritised

  • 🔴 Critical — crashes, data issues, important problems
  • 🟡 High — key features not working properly
  • 🟢 Medium — smaller bugs or UI issues
  • 💬 Low — suggestions and improvements

A Quick Note

This is a small, personal project, so updates happen steadily rather than instantly — but everything you share genuinely helps make the app better 💜
