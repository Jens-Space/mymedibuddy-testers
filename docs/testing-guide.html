Testing Guide 🧪

Here's what we're focusing on in this beta. Your feedback helps us build a better app!

Core Features to Test

  • ✓
    Medication Reminders

    Set reminders and receive notifications at the right time.

  • ✓
    Medication List

    Add, edit, and manage your medications with dosage info.

  • ✓
    History Log

    Track when you've taken each medication.

Test Scenarios

  1. Add a new medication

    Try adding a medication with name, dosage, and frequency.

  2. Set a reminder

    Create a daily reminder and check if you get notified.

  3. Log a dose

    Mark a medication as taken and check the history.

  4. Edit/delete medication

    Modify an existing medication or remove it.

  5. Navigate around

    Try moving between screens and see if everything flows well.

What to Look For

  • Crashes — Does the app freeze or close unexpectedly?
  • UI issues — Are buttons easy to tap? Is text readable?
  • Notifications — Do reminders appear on time?
  • Data accuracy — Are your medications and history saved correctly?

How to Report Issues

Found something? Use the in-app feedback form to let us know. Try to include:

  • What you were doing when the issue occurred
  • What happened vs. what you expected
  • A screenshot if possible

Go to Settings → Send Feedback inside the app.
